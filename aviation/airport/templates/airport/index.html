  • POST http://localhost:8000/add add a new airport to the list of airports
    • http://localhost:8000/add
  • GET http://localhost:8000/airport/[airport_id]/ returns a selected airport and its description, [airport_id] is a placeholder for the specific ID of the airport
    • http://localhost:8000/airport/3/
  • GET http://localhost:8000/airport/ returns a list of all airports
    • http://localhost:8000/airport/
  • UPDATE http://localhost:8000/update/[airport_id]/ updates the information of a selected airport, [airport_id] is a placeholder for the specific ID of the airport
    • http://localhost:8000/update/9/
  • DELETE http://localhost:8000/delete/[airport_id]/ delete a selected airport from the list, [airport_id] is a placeholder for the specific ID of the airport
    • http://localhost:8000/delete/9/
  • API http://localhost:8000/api/
    • http://localhost:8000/api/
